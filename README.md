# Bank Management System (Java)

A simple Bank Management System implemented in Java using Object-Oriented Programming (OOP) concepts. This project is menu-driven and allows users to perform basic banking operations such as deposit, withdrawal, transfer, and viewing account details.

## Features

- Deposit Money  
- Withdraw Money  
- Transfer Money Between Accounts  
- View Account Details  
- Exit Application  

## Concepts Used

- Abstract Class  
- Inheritance  
- Method Overriding  
- Encapsulation  
- Menu-Driven Programming  
- User Input using Scanner  

## Structure

- Account: Contains common properties and abstract methods  
- BANK: Implements deposit and withdraw methods  
- Main: Handles menu and user interaction 

## How It Works

1. Program displays a menu  
2. User selects an option  
3. Corresponding operation is performed  
4. Program continues until user exits
